# TeBATDrift
Garfield++-based TPC electron drift and gas-transport simulation framework for TeBAT, with versioned subdirectories for reproducible development and validation.
=======

Versioned Garfield++/Magboltz simulation framework for TeBAT TPC electron drift, gas transport, and resolution studies.

## Repository structure

- `TeBATDrift_v1.0.0/` : first validated Garfield++ drift simulation release
- future versions will be added as sibling subdirectories
